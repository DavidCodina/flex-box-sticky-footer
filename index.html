Header


This example is intended to demonstrate the use of a flex container for layout purposes. More specifically, it demonstrates how one can create a simple sticky footer. Notice how main has flex: 1;, and body has:


  body {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
    ...
  }

This means that main will take all of the available space not used by the header and footer. Thus if there's not enough content to fill the browser height, main will still stretch. In this case, we have more than enough content to fill the browser height due to the length of this article. However, you can hide the article to see the effect.


  body {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
    ...
  }

  header {
    ...
  }


  main {
    flex: 1;
    ...
  }

  footer {
    ...
  }
